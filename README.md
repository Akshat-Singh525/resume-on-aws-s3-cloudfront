# AWS Resume Website Hosting

## Project Overview
This project demonstrates static website hosting using AWS S3 and CloudFront.

## AWS Services Used
- Amazon S3
- Amazon CloudFront
- GitHub

## Features
- Static website hosting
- HTTPS enabled
- Global content delivery using CDN
- Public access configuration

## Live Website
(https://d15syn849y4blx.cloudfront.net/)

## Architecture
User → CloudFront → S3 Bucket
